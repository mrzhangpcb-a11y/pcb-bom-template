
# PCB BOM Template — Free Download for Electronics Engineers

A ready-to-use **Bill of Materials (BOM) template** for PCB design and assembly projects.
Suitable for SMT, THT, and mixed-technology assemblies.

## 📋 Template Includes

- Component reference designator (RefDes)
- Part number / manufacturer part number (MPN)
- Component description and value
- Package / footprint
- Quantity per board
- Preferred supplier & supplier part number
- Unit cost & extended cost
- Assembly notes (SMT / THT / DNP)
- RoHS compliance field

## 🚀 How to Use

1. Download the template file from this repository
2. Fill in your component information for each row
3. Submit the BOM together with your Gerber files to your PCB assembly manufacturer
